자바스크립트 객체의 유형

자바스크립트에서는 사용자가 직접 객체를 만들어 사용할 수 있지만, 브라우저에서는 많은 객체들을 지원하기 때문에 이를 사용하는 방법도 있다. 브라우저가 제공하는 객체는 코어 객체, HTML DOM 객체, 브라우저 관련 객체(BOM)으로 크게 3가지 유형이 있다.
이 중에서 코어객체를 우선으로 공부해 보았다.



코어 객체의 종류

코어 객체란 자바스크립트가 실행되는 어디서나 사용 가능한 객체들로 여러가지 타입들이 존재한다.
  • Array
  • Math
  • String
  • Date


new 키워드로 코어 객체 생성

c언어와 같이 새로운 객체를 생성할 때 new 코드를 사용하여 생성한다.

var today = new Date( );
var msg = new String('Hello World');
